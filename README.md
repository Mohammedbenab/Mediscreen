# Mediscreen Application

# Introduction
This project is an application for a medical center. The client needs management tool for check patients health status.

 ```bash

 ```
2 - Creat .jar files
 ```bash
   clean mvn install <name-service>
  ```

3 - Creat contener with Docker
